# electron-app-and-TDD

将 Electron 和 Vue 相结合来构建跨平台的桌面应用，结合单元测试。

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
